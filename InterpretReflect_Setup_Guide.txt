================================================================================
1. PREREQUISITES
================================================================================
- Node.js installed (v16 or higher)
- Supabase account (free tier is fine)
- Stripe account (for payments - optional initially)
- Google Cloud Console access (for Google OAuth)
- Apple Developer account (for Apple Sign-In - optional)

================================================================================
2. ENVIRONMENT SETUP
================================================================================
Create a .env file in the /project directory with these variables:

VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_STRIPE_PUBLISHABLE_KEY=your_stripe_publishable_key

================================================================================
3. SUPABASE CONFIGURATION
================================================================================
1. Go to your Supabase Dashboard
2. Enable Authentication Providers:
   - Navigate to Authentication ‚Üí Providers
   - Enable Email (for magic links)
   - Enable Google OAuth:
     - Add Google Client ID and Secret
     - Set redirect URL: https://your-project.supabase.co/auth/v1/callback
   - Enable Apple (if you have Apple Developer account)

3. Configure Email Templates:
   - Go to Authentication ‚Üí Email Templates
   - Customize the Magic Link template for your brand

4. Run Database Migrations:
   - Go to SQL Editor in Supabase
   - Run these migration files in order:
     1. user_profiles_schema.sql
     2. growth_insights_tables_fixed.sql
     3. subscriptions_schema.sql

================================================================================
4. GOOGLE OAUTH SETUP
================================================================================
1. Go to Google Cloud Console (https://console.cloud.google.com/)
2. Create a new project or select existing
3. Enable Google+ API
4. Create OAuth 2.0 credentials:
   - Add authorized redirect URIs:
     - http://localhost:5173
     - https://your-domain.com
     - Your Supabase callback URL
5. Copy Client ID and Secret to Supabase
